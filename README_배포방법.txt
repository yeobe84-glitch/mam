MAM 온라인 V60 GitHub Pages 업로드용

1) 이 ZIP을 압축 해제합니다.
2) mam GitHub 저장소에 내용물을 전부 업로드합니다.
3) Settings > Pages > Branch: main / root 저장.
4) 주소: https://yeobe84-glitch.github.io/mam/

주의: 저장/불러오기는 Render API(mam-api)가 배포되어 있어야 완전히 동작합니다.
기본 API 주소는 https://mam-api.onrender.com 입니다. Render 주소가 다르면 브라우저 콘솔에서 localStorage.setItem("MAM_API_BASE","새주소") 후 새로고침하세요.
